# PROJECT RULES — Personal Loan Tracker App

## Project Goal
Build a minimal premium iPhone app for tracking personal interest-only loans.

Core philosophy:
"Open the app and instantly know who your money is."

## Platform
- iPhone only
- React Native + Expo
- TypeScript
- Expo Router
- Offline-first
- Local SQLite database
- Dark mode only

## Tech Stack
- Expo
- React Native
- TypeScript
- Expo Router
- NativeWind
- Zustand
- Expo SQLite
- Reanimated
- Expo Haptics

## Build Strategy
Build phase-by-phase only.

Do not generate the entire app at once.

## Do Not
- Do not build backend
- Do not add authentication
- Do not add cloud sync
- Do not over-engineer
- Do not create full screens before requested
- Do not change business rules without approval

## Coding Rule
Before writing or modifying code:
1. Explain implementation plan
2. List files to create/change
3. Identify possible risks
4. Wait for approval

## UI Direction
- Dark minimal
- Pastel finance colors
- Large cards
- Smooth iOS feel
- Premium productivity app style
- Bottom sheets for primary actions
- Haptic feedback
- Minimal friction
